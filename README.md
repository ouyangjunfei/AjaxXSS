**执行原理**

返回内容也是一段JS代码，而且会被立即执行，执行结果会输出在正常访问用户的控制台内

由于执行者是正常页面所在的浏览器，所以可以访问当前的cookies，进而再通过一次Ajax调用`post()`发送得到的cookie信息给监听服务器URL`http://127.0.0.1:5000/fake/`
